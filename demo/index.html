Quick Grid version 0.1.0

Stop the madness, ditch the PSD and Illustrator grid templates. You really don't want to open up a graphics suite every time you want to use a grid...right?

The goal for Quick Grid is to eliminate the need for complicated math and cumbersome html markup. Quickly add a responsive grid that you can modify dynamically on the fly -- how's that for control. More details about features available here

This grid system is based on the 1000px grid inspired by an article written by Elliot Stocks and a tutorial written by Steve Hickey.
